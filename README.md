
                             CORALINEOS
                             VERSION 1.0


![CoralineOs Screenshot](preview.png)

[ DESCRIPTION ]
CoralineOS is a Python-based terminal interface designed for 
simplicity and style. It provides a calculator and dices, wrapped in 
a custom-designed ASCII aesthetic.

[ FEATURES ]
1. REAL-TIME CLOCK: Displays the current system time at launch.
2. MULTI-FUNCTION CALCULATOR: 
   - Addition, Subtraction, Multiplication, Division.
   - Power/Exponentiation calculations.
3. DIGITAL DICE: A random number generator (1-6) for gaming.
4. DYNAMIC UI: Uses the 'Rich' library for red-themed styling.

[ HOW TO START ]
1. Ensure Python 3.x is installed.
2. Install dependencies: 'pip install rich'
3. Open the .py file

[ CONTROLS ]
- Use numbers (1 or 2) to navigate the menu.
- Follow the '>' prompts to input data.
- Press 'Enter' to return to the main menu.
